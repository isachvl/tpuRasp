# Telegram Schedule Bot for TPU

## О проекте

Telegram-бот для студентов Томского политехнического университета (ТПУ), позволяющий быстро получать расписание занятий, отслеживать его изменения и создавать персональные напоминания.

Проект автоматически получает актуальное расписание с сайта ТПУ, обрабатывает его и предоставляет удобный интерфейс через Telegram.

---

## Основные возможности

* Просмотр расписания на текущий день
* Навигация по предыдущим и следующим дням
* Просмотр расписания на прошлую, текущую и следующую недели
* Создание напоминаний
* Автоматическая проверка изменений расписания
* Ответы на вопросы пользователя с использованием языковой модели и данных расписания
* Встроенная инструкция по использованию бота

## Структура проекта

```
bot.py          # Основной Telegram-бот
scraper.py      # Загрузка и обработка расписания
ai.py           # Работа с ИИ
data/           # Сохранённое расписание по неделям
```

---

## Как запустить

1. Установить зависимости

```
pip install -r requirements.txt
```

2. Запустить загрузку расписания

```
python scraper.py
```

3. Запустить бота

```
python bot.py
```
